## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/imarwamk/flutterUI.git](https://github.com/imarwamk/flutterUI.git)
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
